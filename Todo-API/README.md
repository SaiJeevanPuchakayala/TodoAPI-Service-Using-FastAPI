## To Run in Local

1. Clone the project
```bash
>_ git clone https://github.com/SaiJeevanPuchakayala/FarmWiser.git
```
2. Install packages from requirements.txt
```bash
>_ pip install -r requirements.txt
```
3. Start server 
```bash
>_ uvicorn index:app --reload
```
4. Access APIs at
```
http://127.0.0.1:8000/docs
```
